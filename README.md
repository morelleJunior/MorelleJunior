## 🧠 Sobre minha atuação

Tenho experiência prática atuando em diferentes etapas do ciclo de desenvolvimento e sustentação de sistemas, desde a análise do problema até a validação final da solução.

### Áreas em que atuo:

* **Banco de Dados:** SQL Server, consultas avançadas, análise de dados, scripts de correção e apoio técnico.
* **Desenvolvimento:** Angular, TypeScript, C#, .NET, WinForms e consumo de APIs.
* **Qualidade de Software:** testes funcionais, regressivos, validação de regras de negócio e análise de bugs.
* **Sistemas ERP:** rotinas fiscais, financeiras, comerciais, estoque, relatórios e integrações.
* **Automação:** criação de ferramentas internas, otimização de processos e redução de tarefas repetitivas.

## 🚀 Projetos em destaque

### 🔹 SQLRápido

Ferramenta desenvolvida para facilitar rotinas técnicas com banco de dados, apoiando consultas, atualizações, backups e automações voltadas para suporte e análise.

**Tecnologias:** `C#` `WinForms` `SQL Server` `Dapper`

**Principais recursos:**

* Conexão dinâmica com banco de dados.
* Execução de consultas SQL.
* Rotinas de atualização.
* Backup automatizado.
* Apoio em análises técnicas.

---

### 🔹 TicketFácil

Sistema para gerenciamento de clientes e tickets, com controle de status e organização de atendimentos técnicos.

**Tecnologias:** `C#` `WinForms` `SQL Server`

**Principais recursos:**

* Cadastro de clientes.
* Controle de tickets.
* Status de atendimento.
* Organização de demandas.
* Apoio ao fluxo de suporte.

---

### 🔹 Pet & Gatô

Aplicação web desenvolvida com foco em gestão de produtos, clientes e autenticação de usuários.

**Tecnologias:** `Angular` `TypeScript` `.NET API`

**Principais recursos:**

* Login de usuário.
* Rotas protegidas.
* Gestão de produtos.
* Gestão de clientes.
* Consumo de API.

---

### 🔹 AvaliaAqui

Projeto web voltado para apresentação comercial, captação de clientes e avaliação de serviços.

**Tecnologias:** `Next.js` `JavaScript` `HTML` `CSS`

**Principais recursos:**

* Página inicial moderna.
* Apresentação comercial.
* Estrutura voltada para conversão.
* Interface responsiva.
* QR Code para acesso rápido.
